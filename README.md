# Introduction

此教程是专门为 Girls Coding Day 女性编程素养工作坊而设计。

工作坊结束后，学员将具备能力在Bootstrap模板的基础上，做出 [Girls Coding Portfolio](https://girlscodingday.org/portfolios.html) 这样的静态网站。

学员将：

1. 具备修改模板的基本HTML和CSS的能力。
2. 了解bootstrap。
3. 部署在Github Page的能力。
4. 自学编程的基本能力。
5. 一群热爱编程的闺蜜以及友好的程序员社群。
