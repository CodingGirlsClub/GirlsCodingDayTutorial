### 引用 CSS

#### 1. HTML 外部样式表

为了让CSS效果能在HTML中显示出来，我们需要在HTML文档中引用CSS文件。引用CSS最好的办法是把所有的样式代码放在额外的一个文件中，在HTML文档中的<head>元素中引用此文件。

**<u>Inquiry Learning</u>** ： 想一想这样引用CSS的好处？

#### 2. 另外两种引用CSS的方法：

- 内部样式表
- 内联样式
